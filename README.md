# BloomWatc
Un proyecto innovador que implementa un modelo U-Net de Deep Learning (PyTorch) para realizar segmentación de imágenes a nivel de píxel con una precisión del 87.24% en la detección de floración. Utiliza datos multiespectrales de la NASA e integra tecnologías geoespaciales como PostGIS y Leaflet/Three.js para una visualización dinámica y accionable.
